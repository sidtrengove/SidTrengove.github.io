Introduction

Throughout my time as an Accountant the use of Excel has been a large part of my job. A significant portion of my usage for Excel was to complete Accounting Workpapers. There are certain parts of this process that were incredibly monotonous, making it a prime candidate for automation.

In this project I have created a set of Working Papers for a Demo Company. The workbook can be used by any accountant to reduce the amount of time they spend on non-value adding actives.
