FINANCIAL DASHBOARD

FINANCIAL INTERCOMPANY INTERACTIVE DASHBOARD USING PYTHON AND POWER BI

Introduction

Producing management reports can be a long-drawn-out process, only to end up with an excel report that is hard to read & understand.

With the powerful combination of Python & PowerBI I’ve built an intercompany dashboard that pulls its data directly from Xero with the click of a button all for FREE!

Endless possibilities

The possibilities for reporting in the way are as endless as the data to hand.

What if you want to see a more detailed breakdown of the items in your Profit & Loss? Simple! Just add another Sheet to the PowerBI Report and throw some handy graphics in. Voilà, an interactive reporting system showing where your money went & where it came from!

New Data

Need to change the date range you’re reporting on? Do I need to know how to write code?

The Report has been built to allow usability; just edit the parameter and let Python pull the data from the Xero for you.
